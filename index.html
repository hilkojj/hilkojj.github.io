Scriptable C++ Game Engine

This game engine focuses on game data and logic.
It partly allows development without recompiling or even restarting your game. This speeds up development a lot.

Features:
  • Multiplayer
  • Saving and loading Levels
  • Live reloading of assets and scripts
  • Lua scripting (entity creation and logic)
  • Entity Inspector (edit entity-data at runtime)
  • Profiler
  • Cross Platform (Linux, Windows, and Web browsers)
